# Hidden Spot Frontend Manual Test Checklist

자동 테스트 도구 없이 프론트엔드 화면/상호작용을 점검하기 위한 수동 테스트 항목입니다.

## 1. 기본 접속
- [ ] 앱 첫 화면이 정상 렌더링된다.
- [ ] 브라우저 콘솔(F12)에 치명적 에러가 없다.

## 2. 환경변수
- [ ] `NEXT_PUBLIC_NAVER_MAPS_CLIENT_ID`가 설정되어 있다.
- [ ] 키 누락 시 지도 영역에 안내 오버레이가 표시된다.

## 3. 지도 로딩/조작
- [ ] 지도 타일이 정상 표시된다.
- [ ] 지도 드래그/줌이 정상 동작한다.
- [ ] 마커가 렌더링된다.

## 4. 검색/필터
- [ ] 검색어 입력 시 리스트가 필터링된다.
- [ ] 점수 필터(min score)가 동작한다.
- [ ] 카테고리 선택 시 결과가 바뀐다.

## 5. 목록-지도 연동
- [ ] 리스트 아이템 클릭 시 지도 중심이 이동한다.
- [ ] 마커 클릭 시 선택 카드가 열린다.
- [ ] "현지도에서 재검색" 버튼 노출/동작이 정상이다.

## 6. 등록/분석 UX
- [ ] URL 입력 후 분석 요청이 전송된다.
- [ ] 로딩 상태가 사용자에게 표시된다.
- [ ] 성공 시 리스트/상세 카드가 갱신된다.
- [ ] 실패 시 에러 메시지가 표시된다.

## 7. 상세 카드
- [ ] 요약/태그/시그니처 메뉴/팁이 표시된다.
- [ ] 카드 닫기/펼치기 동작이 정상이다.

## 8. 모바일 반응형
- [ ] 모바일에서 지도/리스트 토글이 정상이다.
- [ ] FAB 버튼(목록/등록)이 정상 동작한다.
- [ ] 입력창/카드/지도 오버레이 겹침 문제가 없다.

## 9. 성능/안정성
- [ ] 새로고침 후 초기 상태가 정상이다.
- [ ] 동일 URL 재요청 시 앱이 깨지지 않는다.
- [ ] 느린 네트워크에서도 로딩 UX가 유지된다.
